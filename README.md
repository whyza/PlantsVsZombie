>>>本项目是学完Java基础后用GUI写的仿植物大战僵尸

##基本完成的一些功能如下

1. 游戏开始定时产生阳光，向日葵定时产	生阳光，种植植物消耗阳光,增加阳光	
2. 产生僵尸,僵尸移动
3. 在地图中种植植物
4. 不同植物发射不同子弹
5. 子弹击中僵尸的判断
6. 僵尸攻击植物
7. 游戏输赢的判断
8. 击杀僵尸可以获得道具
